Art of Illusion 2.9 Installer for Linux

Introduction

This installer will install Art of Illusion on a Linux system. Please carefully read instructions concerning installation of extensions such as the Java Media Framework.

Contents

The installer provides installation of:

AoI base system
All you need to run Art of Illusion on a standalone basis. The following important features are not included with the base system and must be specified at install time
Java OpenGL (JOGL)
Provided the video drivers you use feature OpenGL, this extension will bring hardware acceleration to 3D modelling. i586 and AMD64 binaries are proposed, but choose only one of the two packages.
Desktop Integration
If you use Gnome/KDE and select desktop integration at install time, AoI files (bearing the .aoi extension) will display a nice icon. Double clicking on the file will open it with Art of Illusion. A menu item will be available in the "Graphics" section of the Applications menu and a desktop icon will be created.

Installation is finished, where to go now ?

Well you can have a nice go at the Hourglass tutorial, a very famous beginner tutorial. And pay a visit to the Scripts and Plugins repository where some very handy extensions are available. Installation of these extensions is very simple : select the Scripts and Plugins Manager from the Tools menu. Select the install tab and choose any script or plugin you want to install. It's as simple as that...
