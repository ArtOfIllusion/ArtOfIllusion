Art of Illusion 2.9 Installation für Windows

Einführung

Dieses Installationsprogramm installiert Art of Illusion auf einem Windows System. Bitte lesen Sie die Installationsanleitung gründlich - insbesondere die über Erweiterungen wie das Java Media Framework.

Inhalt

Das Installationsprogramm bietet folgende Installationsoptionen an:

AoI Basissystem
Alles was man benötigt um Art of Illusion allein zu betreiben. Die folgenden wichtigen Erweiterungen (Features) sind nicht im Basissystem enthalten und sollten während der Installation ausgewählt werden
Java OpenGL (JOGL)
Diese Erweiterung ermöglicht OpenGL Hardware-Beschleunigung für 3D Bearbeitung zu nutzen. Es werden i586 und AMD64 binaries unterstützt, aber bitte wählen Sie nur eines der beiden Pakete aus.
Desktop Integration
Art of Illusion wird mit dieser Option im Startmenü installiert - zusätzlich wird ein Desktop Icon erstellt.

Die Installation ist abgeschlossen - wie geht es weiter?

Nun - Sie können einen Versuch starten das "Stundenglass" Tutorial nachzuarbeiten - eines der beliebtesten Beginner Tutorials. Und besuchen Sie das Scripts und Plugins repository wo einige wirklich nützliche Erweiterungen verfügbar sind. Die Installation dieser Erweiterungen ist einfach: Den Script- und Pluginmanager aus dem Werkzeug Menü auswählen, das Installations-Tab auswählen - danach einfach jedes beliebige Script oder Plugin zwecks Installation aussuchen.
